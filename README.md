# Turkish BSL American Players Dashboard

Track American basketball players in the Turkish Basketbol Super Ligi.

## Quick Start

```bash
pip install -r requirements.txt
python daily_scraper.py
python hometown_lookup_fixed.py
python join_data.py
python dashboard.py
```

Open http://localhost:5000

## Deploy to Render

Uses Docker. Connect repo and deploy.

## Data Source

TheSportsDB API (League ID: 4475)
